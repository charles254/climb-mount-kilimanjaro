# Scientific On-Page SEO Checklist

Follow these steps for every target keyword based on the SEO Fight Club Methodology.

### Phase 1: Benchmark the Leaders
- [ ] List the top 3-5 organic ranking URLs.
- [ ] Run `analyze_onpage.py` to get their baseline metrics.
- [ ] Calculate the **Average Winner Baseline**:
    - Average Word Count: _______
    - Average Keyword Density: _______%
    - Average Image Count: _______

### Phase 2: Eliminate "Zeros"
Check your page against the winners. Do you have a "Zero" in any of these?
- [ ] H1 Tag (Does content match keyword?)
- [ ] H2 Tags (Are they used? How many?)
- [ ] Bold / Strong text for emphasis.
- [ ] Image Alt text containing the entity.
- [ ] FAQ Schema / Structured Data.

### Phase 3: Entity Expansion
- [ ] Identify 5-10 related entities used by competitors.
- [ ] Naturally weave these into paragraphs, H3s, or Image Captions.
- [ ] Check if the internal linking structure matches the leader's depth.

### Phase 4: Technical "Source" Tuning
- [ ] View Page Source.
- [ ] Check total keyword occurrences in HTML (including non-visible area).
- [ ] Adjust metadata and aria labels to match the leader's total "signal volume."
